# old-press · image-generation prompts

## Material suffix (always append, any theme)

```
halftone, bold halftone dots, screen printing, screen print texture, letterpress,
old newspaper texture, raw newsprint paper, aged paper, paper grain, low-fidelity
printing, ink bleed, misregistered ink, uneven ink coverage, vintage print
```

## Gaslight theme (default)

Theme words:

```
1920s London, gaslight, fog, Victorian architecture, typewriter, brass, pocket
watch, warm darkness, engraving, etching, editorial layout, swiss grid, strong
hierarchy, negative space, newspaper advertisement layout, weathered typography,
cream paper, charcoal black, dark brown, brass gold, forest green, deep burgundy,
sepia
```

References to name-drop: Sherlock Holmes, Spider-Man Noir, The Third Man, The
Maltese Falcon, Peaky Blinders, British Museum, old Scotland Yard.

Negative prompt (**Gaslight only** — do not carry into other themes):

```
no 1980s, no neon, no cyberpunk, no synthwave, no vaporwave, no futuristic,
no plastic materials, no high saturation, no cartoon style, no Y2K,
no skeuomorphic UI, no bright RGB lighting, no modern tech aesthetic,
no glossy digital finish, no clean vector perfection, no perfect ink registration
```

## Vapor theme (alternate)

```
90s vaporwave motifs screen-printed on aged newsprint, muted lavender cyan and
magenta inks, desaturated, printed not glowing, halftone grid instead of neon grid
```

Keep the material suffix; drop the neon bans (obviously), keep
`no glossy digital finish, no perfect ink registration`.

## One-line style definition

> Old London Editorial Noir — a quiet personal archive rendered through interwar
> British newspaper advertising, bold halftone printing, aged newsprint, engraved
> London architecture, warm darkness, and restrained editorial typography.
